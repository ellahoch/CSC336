Machine Learning Types and Algorithms

Supervised Learning

  1. Categorical Target Variable
    • Regression
      Ex. Housing Price Prediction
      1. Linear Regression
      2. Logistic Regression
  2. Continuous Target Variable
    • Classification
      Ex. Medical Imaging
      1. Decision Trees
      2. Ensembles
        • Random Forest
        • Boosting
        • Bootstrap

Unsupervised Learning

  • Target variable not available
    1. Clustering
      Ex. Customer Segmentation
    2. Association
      Ex. Market Basket Analysis

Semi-supervised Learning

  • Categorical Target Variable
    1. Classification
      Ex. Text Classification
    2. Clustering
      Ex. Lane-finding on GPS data

Reinforcement Learning

  1. Categorical Target Variable
    • Classification
      Ex. Optimized Marketing
  2. Target Variable Not Available
    • Control
      Ex. Driverless Cars
